# AutoResearch Foundry

> **This lab is hosted externally.** View the full lab at: [ejones18/autoresearch-foundry](https://github.com/ejones18/autoresearch-foundry)

An autonomous AI research agent that iteratively improves ML training scripts. Built on Azure AI Foundry's Hosted Agent platform with a background worker architecture that plans experiments via LLM, submits Azure ML training jobs, and accepts or rejects changes based on metric improvements.
